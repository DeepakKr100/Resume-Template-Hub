# Frontend Environment Files

## Local Development (.env)

```env
REACT_APP_API_URL=http://localhost:5080/
REACT_APP_RAZORPAY_KEY=rzp_test_9ta1IN9vcYLKN5
```

## Production (.env.production)

```env
REACT_APP_API_URL=https://your-render-backend.onrender.com/
REACT_APP_RAZORPAY_KEY=rzp_live_xxxxx
```

Update `your-render-backend` with your actual Render service name.
Update `rzp_live_xxxxx` with your production Razorpay key.
